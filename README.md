## üîß Development

### Prerequisites
- Node.js 18+
- npm or yarn
- Git

### Getting Started
```bash
# Clone the repository
git clone https://github.com/nvtank/TripC-Solutions.git

# Navigate to project directory
cd TripC-Solutions/project

# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```
